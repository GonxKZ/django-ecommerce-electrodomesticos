# E-commerce de Electrodomésticos con Django

Este es un proyecto básico de e-commerce de electrodomésticos, construido con Django y configurado para usar Docker y opciones de despliegue en PythonAnywhere o Render.

## Requisitos

1. **Python 3.8 o superior**
2. **Django 3.0 o superior**
3. **Docker** (Opcional pero recomendado)
4. **Podman** (Alternativa a Docker)
5. **Stripe API Key** (para integración de pagos en Sprint 2)
